SKILLS

HTML

・基本的なタグの使い方や目的・文法を理解しサイトの構造によって使い分けることができます。

・CSS設計のFLOCSSやBEMの基本を学び管理のしやすいコードを書くことができます。

・インデントやコメントアウトを使い、他人が見ても分かりやすいコーディングを意識しています。

・画像ファイルの最適化なども忘れないように意識しています。

CSS

・FlexboxやGrid、ボーダーや背景、テキストスタイルの設定などの基本的なCSSを理解しています。

・メディアクエリを使ったレスポンシブデザインの実装ができ、画面幅を変えても崩れないように工夫することができます。

・ホバー時やアクティブ時などの動きをキーフレームアニメーションやトランジションを使って実装することができます。

・フォームのチェックボックスやラジオボタンなどのよく使われる動きを実装できます。

JavaScript / jQuery

・jQueryを中心に、CDNなどを利用してモーダルやアコーディオン、スライダーなどを実装することができます。

・特定の値を取得したり、それを利用して簡単なコードを書いてサイトに動きをつけることができます。

・WEB制作でよく使われる基礎的な動きは大体は実装することができます。

・JavaScriptの知識はまだ乏しいので、複雑なコードを1から書くことは難しいです。

WordPress

・コーディングした静的サイトを使ってWordPressのオリジナルテーマを作成することができます。

・テンプレートタグを使ってサイト内の情報を取得して動的にすることができます。

・固定ページやアーカイブページなどの基本的なページに加え、カスタム投稿・カスタムタクソノミー・カスタムフィールドを使うことができます。

・サイトをローカル環境から本番環境へ移行したりメタタグやOGPタグの設定ができます

・プラグインを使ってサイトの引っ越し・サイトや画像の最適化・セキュリティ対策などを行うことができます。

Sass

・基本的なSassの使い方を理解しており、ネストやmixin、変数、メディアクエリなども使用してできるだけ読みやすいコードを書けるように意識しています。

・FLOCSSやオブジェクト指向の基本を理解しており、レイヤー毎にファイルの分割して管理することができます。

Gulp

・Sassファイルを自動的にコンパイルさせたり、watchを使ってcssを自動で反映させることができます

・HTMLファイルを自動でリロードして効率的にコーディングすることができます。

・ベンダープレフィックスを自動でつけたり

・CSSファイルやJSファイルを圧縮することができます。

Pug

・Pugを使って省略したコードを書くことができます。

・共通部分をテンプレート化したり変数を使うことができます。

Figma

・デザインカンプから値を取得してコーディングすることができます

・ツールやレイヤーなどの基礎的な使い方を理解しており、簡単なデザインやデザインカンプを作ることができます。
